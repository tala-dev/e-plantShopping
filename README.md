## Overview

Paradise Nursery is a frontend shopping application built with **React** that allows users to browse a selection of house plants and manage a shopping cart. The project demonstrates core React concepts such as component-based architecture, state management, and client-side routing.

The application consists of three main pages: a landing page, a product listing page, and a shopping cart page.

---

## Features

- Browse a catalog of house plants  
- Add products to a shopping cart  
- Increase or decrease item quantities  
- View the total cost of items in the cart  
- Navigate between pages without full page reloads  

---

## Application Pages

### Landing Page
- Entry point of the application  
- Provides a brief introduction to Paradise Nursery  
- Includes navigation to the product listing page  

### Product Listing Page
- Displays available house plants  
- Each product can be added to the shopping cart  
- Uses React state to track selected items  

### Shopping Cart Page
- Displays all items added to the cart  
- Allows quantity adjustments before checkout  
- Automatically recalculates the total price  

---

## User Flow

1. User opens the landing page  
2. User navigates to the product listing page  
3. User adds plants to the shopping cart  
4. User reviews and updates the cart  
5. User sees the final total cost  

---

## Technologies Used

- **React** (functional components and hooks)  
- **React Router** for page navigation  
- JavaScript (ES6+)  
- HTML5  
- CSS  

---

## State Management

- Application state is managed using **React hooks**  
- Cart state is shared across pages to ensure consistency  
- Quantity and total price updates are handled dynamically  

---

## Project Scope

- Frontend-only React application  
- No backend or payment processing  
- Focus on UI logic, state updates, and component interaction
